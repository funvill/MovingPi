# MovingPi
A python script for a Raspberry PI that plays a sound effect when it is moved. 

Hardware
-------------

- Raspberry pi 3
- MPU-6050 - SparkFun Triple Axis Accelerometer and Gyro Breakout 

Sound effect 
-------------
